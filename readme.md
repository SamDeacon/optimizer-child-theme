# Optimizer theme - wordpress child theme boilerplate

## Includes
- style.css (compiled from sass)
- functions.php enqueuing scripts/stylesheet override
- screenshot.png example

## Gulp build tool
- gulp-sass
- auto-prefixer
- gulp watch for auto-compile
- stored in 'scss' folder
### to run, commandline:
- npm install -g gulp
- npm install
- gulp
